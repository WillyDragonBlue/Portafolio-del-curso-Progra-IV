Hola mundo al Bootstrap

¿Qué es Bootstrap?

Es un Framework o un conjunto de herramientas que sirben para crear interfaces web compatibles con CSS y JavaScript, es uno de los reponsables del diseño responsivo.

¿Para que se usa Bootstrap?

El diseño responsive o sitio web adaptativo lo que permite que Bootstrap sea una herramienta para adaptar los sitios web con interfaces

dependiendo la resolución de la pantalla adaptandose al espacio, funciona como un contenedor de contenidos.
